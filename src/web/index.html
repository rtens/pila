PILA PHP Isolation Library Assessment

The table below show dynamically assessed qualities of PHP isolation libraries. The qualities are inspired by Roy Osherove's list of Fifteen things I look for in an Isolation framework.

If you would like to add another library of feel like one of the library wasn't assessed correctly, please don't hesitate to contact me or create a pull request.

Check out the project on github

Full disclosure: I'm the author of mockster so it might seem suspicious that it happens to achieve the highest score. I tried to be fair but a part of this assessment was to show why I wrote mockster and to start a discussion about the quality of isolation libraries so I appreciate any comment.
